# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users

- `GET /api/users/:id`
- `POST /api/users`
- `PATCH /api/users/:id`
- `DELETE /api/users/:id`

### Session

- `POST /api/session`
- `DELETE /api/session`

### Images
- `GET /api/images`
- `GET /api/images/:id`
- `POST /api/images`
- `DELETE /api/images/:id`

### Comments
- `GET /api/comments`
- `POST /api/comments`
- `DELETE /api/comments/:id`

### Likes
- `GET /api/likes`
- `POST /api/likes`
- `DELETE /api/likes/:id`

### Follows
- `GET /api/follows`
- `POST /api/follows`
- `DELETE /api/follows/:id`
